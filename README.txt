Animal Animation Framework - Source

This folder contains the source code for the RimWorld mod "Animal Animation Framework".
To build:
1. Copy Assembly-CSharp.dll and UnityEngine.dll from your RimWorld installation into References/.
2. Open AnimalAnimationFramework.sln in Visual Studio.
3. Build in Release mode. The resulting DLL will be in bin/Release/ and can be copied to the mod's Assemblies/ folder.
